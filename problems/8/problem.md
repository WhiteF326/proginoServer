### 問題文

正の整数 $n$ が与えられます。
$s = 1^2, 2^2, 3^2, \ldots, n^2$ とします。
$s$ を求めてください。

### 制約

- $1 \leq a, b \leq 10^3$

---

### 入力

入力は以下の形式で標準入力から与えられる。

```
n
```

### 出力

$s$ を出力して、改行してください。

---

#### 入力例 1

```
7
```

#### 出力例 1

```
140
```

#### 入力例 2

```
18
```

#### 出力例 2

```
2109
```

#### 入力例 3

```
1000
```

#### 出力例 3

```
333833500
```
