### 問題文

正の整数 $a, b$ が与えられます。
$a, b$ の bitwise XOR を求めてください。

### 制約

- $1 \leq a, b \leq 10^9$

---

### 入力

入力は以下の形式で標準入力から与えられる。

```
a b
```

### 出力

答えを出力して、改行してください。

---

#### 入力例 1

```
7 8
```

#### 出力例 1

```
15
```

#### 入力例 2

```
7 4
```

#### 出力例 2

```
3
```

#### 入力例 3

```
12 24
```

#### 出力例 3

```
20
```
