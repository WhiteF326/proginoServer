### 問題文

正の整数 $n$ が与えられます。
$n$ を $2$ 進数で表し、頭に余分な $0$ を付けずに出力してください。

### 制約

- $1 \leq n \leq 10^9$

---

### 入力

入力は以下の形式で標準入力から与えられる。

```
n
```

### 出力

答えを出力して、改行してください。

---

#### 入力例 1

```
19
```

#### 出力例 1

```
10011
```

`0010011` などの出力は認められません。

#### 入力例 2

```
7
```

#### 出力例 2

```
111
```

#### 入力例 3

```
99124989
```

#### 出力例 3

```
101111010001000011011111101
```


#### 入力例 4

```
1
```

#### 出力例 4

```
1
```
