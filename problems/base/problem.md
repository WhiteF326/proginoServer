### 問題文

$a÷b$ の商と余りを出力してください。

### 制約

- $a, b$ は $1$ 以上 $1000$ 以下の整数

---

### 入力

入力は以下の形式で標準入力から与えられる。

```
a b
```

### 出力

商を $Q$ とし、余りを $R$ とすると、

```
Q R
```

というように空白で区切って出力し、改行してください。

---

#### 入力例 1

```
9 4
```

#### 出力例 1

```
2 1
```

#### 入力例 2

```
19 6
```

#### 出力例 2

```
3 1
```

#### 入力例 3

```
4 5
```

#### 出力例 3

```
0 4
```

商が $0$ である場合もあります。


#### 入力例 4

```
15 5
```

#### 出力例 4

```
3 0
```

余りが $0$ である場合もあります。
